# 🌤 Weather App

A simple web application to check the current weather by city name.  
Built with **HTML**, **CSS**, and **JavaScript** using the **OpenWeatherMap API**.

## 🚀 Features
- Search weather by city name  
- Display weather information  

## 🛠 Technologies
- HTML5  
- CSS3  
- JavaScript (ES6+)  
- OpenWeatherMap API
